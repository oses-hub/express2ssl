<h1 align="center">express2ssl</h1>

<div align="center">

Simple Package to redirect from http to https, with express server!


</div>

## Install

```sh
npm i express2ssl

Or

npm install express2ssl
```

## Usage

```js
const express2ssl = require('express2ssl');
const express = require('express');
const app = express();

app.use(express2ssl());

// Your app code....
```
